# 문제
[구구단](https://www.acmicpc.net/problem/2739)  
간단한 반복문, 문자열 포매팅 문제입니다.  

# 풀이
파이썬에 있는 f-string이라는 기능을 이용해 포매팅하면 됩니다.

# 코드
```py
n = int(input())

for i in range(1, 10):
    print(f"{n} * {i} = {n*i}")
```
